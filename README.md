# Brazilian E-Commerce Dashboard

## Setup environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install pandas scipy matplotlib seaborn jupyter streamlit
```

## Run streamlit app
```
cd dashboard
streamlit run dashboard.py
```
